# Noom (Zoom Clone)
실시간 화상 채팅 서비스.

#WebRTC #Websockets #Node.js #Express #Pug

## Features
* Chat room (Create/Enter/Exit)
* Nickname
* Audio/Video
* Realtime Chatting (Data Channels)
